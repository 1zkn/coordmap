Simple Map Embedding

Embed maps anywhere with a simple URL

Quick Usage

map.zkn.app/o/{latitude},{longitude}
map.zkn.app/o/41.9028,12.4964

Just add coordinates to the URL and get an instant map view

Embed in Website

<iframe src="https://map.zkn.app/o/41.9028,12.4964" width="600" height="450" allowfullscreen loading="lazy"></iframe>

Simple iframe implementation for any website
